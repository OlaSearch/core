#### Usage 

```html
import { Fields } from '@olasearch/core'
const { Button } = Fields
```

```js
<Button label='My button' />
```

As a text link

```js
<Button
  label='My button'
  textLink
  url='//olasearch.com'
/>
```

Full width button

```js
<Button fullWidth>Heya</Button>
```

With search result
```js
<Button url='//olasearch.com'>Read more</Button>
```
