#### Usage

```js
const result = {
  title: 'This is a title',
  url: 'https://cdn.preterhuman.net/texts/science_and_technology/artificial_intelligence/PROLOG%20PROGRAMMING%20FOR%20ARTIFICIAL%20INTELLIGENCE%20-%20lvan%20Bratko.pdf'
};
<Url
  result={result}
/>
```

With custom label
```js
const result = {
  title: 'This is a title',
  url: 'http://www.olasearch.com/a-really-long-url'
};
<Url
  result={result}
  label='Hey click here'
/>
```
